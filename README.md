# 世界围棋发展史｜Go Presentation

一个关于**世界围棋发展史**的交互式网页展示项目，内容涵盖围棋的起源、传播、规则演变，以及著名棋手的生平与时代影响。

## 项目简介

这个项目以网页演示的方式梳理围棋的发展脉络，适合用于：

- 历史与文化主题展示
- 围棋知识科普
- 课程汇报 / 演讲展示
- 个人前端作品集展示

## 内容结构

本项目主要包括以下几个部分：

- 围棋的起源与早期发展
- 围棋在中国、日本、韩国的发展历程
- 围棋规则的演变
- 世界围棋现代化与国际化
- 著名棋手生平信息
- AI 时代对围棋的影响

## 项目亮点

- 以“世界围棋发展史”为主题的可视化展示
- 适合做演讲、展示页或知识型前端项目
- 内容兼顾历史脉络与规则演变
- 适合作为文化类 / 教育类作品集项目
